# Lesson: Chaining If Else Statements
`if/else` statements can be chained together for complex logic. Here is `pseudocode` of multiple chained `if` / `else if` statements:

```js
if(condition1) {
  statement1
} else if (condition2) {
  statement2
} else if (condition3) {
  statement3
. . .
} else {
  statementN
}
```
